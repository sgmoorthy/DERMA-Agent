Overview

What is DERMA-Agent?

DERMA-Agent is an autonomous AI framework for cancer pathology research that combines knowledge graphs, parallel hypothesis testing, and advanced image analysis to accelerate scientific discovery.

🎯

Autonomous Discovery

AI-driven hypothesis generation and testing with self-correcting code execution. Tests multiple hypotheses in parallel using real clinical data.

🧠

Knowledge Fabric

Medical knowledge graph with 50+ nodes (genes, drugs, pathways, cancers). Guides hypothesis generation with evidence-based relationships.

📊

TCGA Integration

Direct access to 20+ cancer cohorts from The Cancer Genome Atlas. Automatic survival data preparation and clinical feature extraction.

🔬

Pathology AI

Advanced segmentation and feature extraction from histopathology images. Detects nuclei, analyzes texture, and generates quantitative metrics.

Tutorial

Step-by-Step Usage Guide

Get started with DERMA-Agent in minutes

📦 Installation

Step 1: Clone the Repository

git clone https://github.com/sgmoorthy/DERMA-Agent.git
cd DERMA-Agent

Step 2: Create Virtual Environment (Recommended)

# Windows
python -m venv venv
venv\Scripts\activate

# macOS/Linux
python3 -m venv venv
source venv/bin/activate

Step 3: Install Dependencies

pip install -r requirements.txt

📋 This installs 30+ packages including: langgraph, langchain-openai, pandas, scikit-learn, lifelines, streamlit, plotly, networkx, opencv-python

Step 4: Set API Key (Optional)

# Windows (PowerShell)
$env:OPENAI_API_KEY="your-api-key-here"

# macOS/Linux (Bash)
export OPENAI_API_KEY="your-api-key-here"

# Or create .env file
echo "OPENAI_API_KEY=your-api-key-here" > .env

⚠️ Without API key, the agent runs in MOCK MODE with limited functionality

Step 5: Verify Installation

python demo_quickstart.py

✅ This runs a quick demo testing all components

Option 1: Run the Quick Demo

python demo_quickstart.py

🧠 Using the Knowledge Fabric

Step 1: Create/Load Knowledge Graph

from tools.knowledge_fabric import create_default_knowledge_fabric

# Create with default oncology knowledge base
kg = create_default_knowledge_fabric("data/knowledge_fabric.json")

# Or load existing
# kg = KnowledgeFabric.load("data/knowledge_fabric.json")

Step 2: Query the Knowledge Graph

# Get statistics
stats = kg.get_statistics()
print(f"Nodes: {stats['total_nodes']}, Edges: {stats['total_edges']}")

# Find drugs treating a specific cancer
drugs = kg.find_paths("BRAF", "Melanoma", max_depth=2)
for path in drugs:
    print(" → ".join([node.id for node in path]))

# Semantic search
cancer_nodes = kg.semantic_search("melanoma skin cancer")
for node in cancer_nodes:
    print(f"{node.id}: {node.properties.get('description', '')}")

Step 3: Add Custom Knowledge

from tools.knowledge_fabric import Node, Edge

# Add custom gene
kg.add_node(Node(
    id="MY_NEW_GENE",
    type="Gene",
    properties={"full_name": "My New Gene", "chromosome": "17"}
))

# Add relationship
kg.add_edge(Edge(
    source="MY_NEW_GENE",
    target="Melanoma",
    relation="MUTATED_IN",
    properties={"frequency": "15%", "source": "custom_study"}
))

Step 4: Save Knowledge Graph

kg.save("data/knowledge_fabric.json")
print("Knowledge graph saved!")

🔬 Running Discovery

Step 1: Configure Discovery

from agents.discovery_engine import DiscoveryConfig

config = DiscoveryConfig(
    max_iterations=3,              # Self-correction attempts
    parallel_workers=4,              # Parallel hypothesis testing
    hypothesis_per_cohort=5,       # Hypotheses per cancer type
    significance_threshold=0.05,     # P-value threshold
    use_knowledge_fabric=True,       # Use knowledge graph
    auto_correct_errors=True,        # Auto-fix code errors
    timeout_seconds=120              # Max execution time
)

Step 2: Run Discovery on Single Cohort

from agents.discovery_engine import FastDiscoveryEngine
from tools.knowledge_fabric import create_default_knowledge_fabric

# Load knowledge graph
kg = create_default_knowledge_fabric("data/knowledge_fabric.json")

# Initialize engine
engine = FastDiscoveryEngine(config, kg)

# Run discovery
results = engine.discover_single_cohort(
    project_id="TCGA-BRCA",
    cancer_name="Breast Cancer"
)

print(f"Tested {len(results)} hypotheses")
print(f"Significant findings: {len([r for r in results if r.significant])}")

Step 3: Run Discovery on Multiple Cohorts

from agents.discovery_engine import run_fast_discovery

# Run on multiple cancer types
results = run_fast_discovery(
    cancer_types=["Breast Cancer", "Skin Cancer", "Lung Cancer"],
    config=config
)

# Results saved to discoveries/discovery_report_*.json

Step 4: Analyze Results

# Print results
for result in results:
    print(f"\nHypothesis: {result.hypothesis[:80]}...")
    print(f"P-value: {result.p_value}")
    print(f"Significant: {result.significant}")
    print(f"Execution time: {result.execution_time:.2f}s")
    if result.figures:
        print(f"Generated {len(result.figures)} figures")

# Save ledger
engine.save_ledger("discoveries/my_discovery.json")

🔍 Pathology Analysis

Step 1: Analyze Image Tile

from tools.enhanced_pathology import TissueAnalyzer, create_synthetic_pathology_image
import matplotlib.pyplot as plt

# Create or load image
image = create_synthetic_pathology_image((512, 512), pattern='mixed')

# Initialize analyzer
analyzer = TissueAnalyzer()

# Analyze tile
features = analyzer.analyze_tile(image)

print(f"Nuclei count: {features.nuclei_count}")
print(f"Cellularity: {features.cellularity:.3f}")
print(f"Tissue area ratio: {features.tissue_area_ratio:.3f}")

Step 2: Segment Nuclei

from tools.enhanced_pathology import NucleiSegmenter

# Initialize segmenter
segmenter = NucleiSegmenter(method='hsv')  # Options: hsv, lab, adaptive

# Segment nuclei
mask = segmenter.segment_nuclei(image)
nuclei_info = segmenter.extract_nuclei_features(image, mask)

print(f"Detected {len(nuclei_info)} nuclei")

# Visualize
fig, axes = plt.subplots(1, 3, figsize=(15, 5))
axes[0].imshow(image)
axes[0].set_title('Original')
axes[1].imshow(mask, cmap='gray')
axes[1].set_title('Segmentation Mask')
axes[2].imshow(image)
axes[2].imshow(mask, alpha=0.3, cmap='Reds')
axes[2].set_title('Overlay')
plt.show()

Step 3: Extract Texture Features

# Texture features are automatically extracted
if features.texture_features:
    texture = features.texture_features
    print(f"GLCM Contrast: {texture.get('contrast', 0):.3f}")
    print(f"GLCM Homogeneity: {texture.get('homogeneity', 0):.3f}")
    print(f"LBP Uniformity: {texture.get('lbp_uniformity', 0):.3f}")
